## Does `source_id` name the identifier's namespace?

A value shared across many sources suggests the namespace is fixed by `type` and that
`source_id` is only provenance; a value essentially never shared suggests the source *is*
the namespace. Read the rates, not the raw counts, and note the confound: types whose IDs
are bare integers (MANUAL_X_REF across 37 sources, REGISTRY_NUMBER across 3) collide by
coincidence, so a few percent means nothing there. The signal is the contrast between CAS
-- 27% of structurally distinctive `nnnnn-nn-n` values shared, which coincidence cannot
explain -- and CITATION's 2 in 106,179 despite PMIDs also being bare integers.
